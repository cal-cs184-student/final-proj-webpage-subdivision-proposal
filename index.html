Implementation of Additional Subdivision Schemes

Summary

In this project, we will be exploring additional subdivision schemes: the Butterfly scheme, the modified Butterfly scheme, Catmull-Clark scheme, the polar subdivision scheme, etc. We will compare the pros and cons for each of them, and discuss the tradeoffs when choosing a subdivision scheme under different scenarios.

Problem Description

Subdividing meshes is useful for rendering a more smooth, detailed surface. Subdivision methods allow us to take a coarse, simple mesh of an object and extrapolate from it to create more smooth edges. In Project 2, we implemented loop subdivision for triangle mesh surfaces. For our final project, our group will implement alternative methods of subdivision. Loop subdivision only works for triangle meshes, so meshes made from quadrilaterals require a different subdivision scheme, such as Catmull-Clark. Even for triangle meshes, different subdivision schemes have tradeoffs. Loop subdivision can yield smoother final surfaces, while Interpolating methods like the butterfly and modified butterfly method maintain key aspects of the original mesh. The challenges of implementing different subdivision schemes lie in the details inherent to meshes: keeping track of vertices and edges within the mesh, creating new mesh elements only when necessary, and updating the mesh correctly with each subdivision. Our team plans to implement different options for subdivision (e.g. Butterfly, modified butterfly, Catmull-Clark, Doo-Sabin) to provide functionality and benefits of these different schemes.
